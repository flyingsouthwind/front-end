
## 相关知识

### Yarn

Yarn 是由 Facebook、Google、Exponent 和 Tilde 联合推出的新的 JS 包管理工具 ，它弥补 npm 的一些缺陷。

相比于 Npm，Yarn 的改善：

* 速度快
  * 并行安装，并行执行包安装
  * 离线缓存，缓存所有下载过的包
* 安全性
  * 每个安装包被执行前校验其完整性
* 版本统一
  * 通过格式详尽而又简洁的 lockfile文件和确定性算法保证安装过程在不同系统时一致的

基于 Yarn 的压力，Npm 从 5.0 版本也开始提升速度以及通过 package-lock.json 文件控制不同系统包安装的一致性。
